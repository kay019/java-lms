# 2단계 - 수강신청(도메인 모델)
## 도메인 설계
### Course
- List<Session>
### Session
- 시작일
- 종료일
- 강의 커버 이미지 (SessionThumbnail)
- 수강료 (무료/유료 구분)
  - 유료일 경우 결제 금액과 수강료가 일치해야 수강 신청 가능
- 최대 수강 인원
  - 무료일 경우 제한 없음
- 상태 (준비중, 모집중, 종료)
  - 모집중일 때만 수강 신청 가능
### SessionThumbnail
- 크기: 1MB 이하
- 타입: gif, jpg(jpeg), png, svg만 허용
- 높이/너비: 300/200px 이상, 비율은 3:2
